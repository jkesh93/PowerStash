Frequently Asked Questions

How do I get a Google API Key?

You'll need a Gemini API key to use the AI features. It's free and easy to get started:

  1. Visit Google AI Studio.
  2. Click the "Get API key" button. You may need to sign in with your Google account.
  3. If prompted, create a new project in Google Cloud.
  4. Click "Create API key in existing project" or "Create API key".
  5. Your new key will be generated. Copy it.
  6. Paste the key into the settings modal in the PowerStash AI app when you first launch it.

Do I need to sign up for PowerStash AI?

No. PowerStash AI requires no user accounts, registration, or login. You can start using it immediately. It's designed to be simple and private.

Where is my data stored?

Locally on your device: Your entire script library and API key are stored in your browser's local storage. PowerStash AI servers do not receive or store any of this information.

When using AI features: To generate, edit, or analyze scripts, the relevant code and your prompts are sent to the Google Gemini API for processing. Please exercise the same caution you would with any AI service and avoid submitting sensitive or proprietary information.

Is PowerStash AI free?

Yes, the PowerStash AI application is 100% free and open-source. However, you are responsible for any costs incurred from using the Google Gemini API. Google provides a generous free tier that is more than enough for most users to get started. You can check their pricing page for details.
